## Featured project — koverai

> **AI-powered test generation CLI** — zero config, published on npm

[![npm](https://img.shields.io/npm/v/koverai?color=0F6E56&label=npm)](https://www.npmjs.com/package/koverai)
[![npm downloads](https://img.shields.io/npm/dm/koverai?color=0F6E56)](https://www.npmjs.com/package/koverai)
[![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)](https://opensource.org/licenses/MIT)

`koverai` auto-detects your framework and generates unit + integration tests using the LLM of your choice — no manual test writing required.

```bash
npx koverai generate
```

**Supports:** React, Vue, Angular, Svelte, Express, NestJS, Spring Boot, Django, Flask, Go and more  
**LLM backends:** Claude (Anthropic) · OpenAI · Gemini — bring your own API key  
**Outputs:** Coverage scores, letter grades, configurable thresholds via `koverai.config.json`
